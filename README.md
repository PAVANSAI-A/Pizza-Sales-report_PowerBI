#  Pizza Sales Report (Power BI)

## 📊 Project Overview

This project analyzes pizza sales data to uncover key business insights using Power BI.

## 📁 Dataset

* 45,000+ records of pizza orders
* Includes order details, pricing, categories, and timestamps

## 🛠 Tools Used

* Power BI
* Excel
* SQL

## 📌 Key KPIs

* Total Revenue: ₹8.17 Lakhs
* Total Orders: 21,350
* Avg Order Value: ₹38.31
* Avg Pizzas per Order: 2.32

## 📈 Key Insights

* Friday recorded the highest number of orders (~3.5K)
* Top 5 pizzas contributed ~25% of total revenue
* Peak order hours identified in evening time
* Classic category generated maximum sales
